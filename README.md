# 🌱 EcoCarbon

## Calculadora de Huella de Carbono

EcoCarbon es una aplicación desarrollada en Python que permite calcular la huella de carbono generada por el consumo de energía eléctrica de hogares o pequeñas empresas.

## Objetivo

Desarrollar una herramienta que ayude a estimar las emisiones de dióxido de carbono (CO₂) producidas por el consumo eléctrico, promoviendo el uso responsable de la energía y la conciencia ambiental.

## Funcionalidades

- Validación de datos ingresados.
- Cálculo de emisiones de CO₂.
- Clasificación del nivel de emisiones.
- Recomendaciones para reducir la huella de carbono.
