# Avance Semana 3

## Proyecto

**EcoCarbon - Calculadora de Huella de Carbono**

## Objetivo del módulo Core

Desarrollar la lógica principal del sistema para calcular la huella de carbono generada por el consumo de energía eléctrica, aplicando buenas prácticas de programación, validaciones de entrada y una arquitectura organizada.

## Avances realizados

Durante esta semana se implementó la estructura inicial del proyecto utilizando GitHub como sistema de control de versiones.

Se desarrolló la arquitectura de carpetas del proyecto:

- src
- tests
- docs
- database

También se crearon los siguientes módulos:

- core.py
- main.py
- validator.py
- recommendations.py

Además, se preparó la carpeta destinada para la documentación y futuras pruebas del sistema.

## Próximas actividades

- Integrar una base de datos.
